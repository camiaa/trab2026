Sobre o Projeto

O sistema Biblioteca Virtual foi desenvolvido com o objetivo de auxiliar no gerenciamento de livros, usuários e empréstimos. O projeto permite cadastrar livros, registrar usuários, controlar empréstimos e visualizar atrasos de devolução. O sistema foi desenvolvido utilizando Django e Python, aplicando Programação Orientada a Objetos com uso de classes.
